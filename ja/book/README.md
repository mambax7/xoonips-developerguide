## Tutorial Purpose 

This document describes the creation process of the item type module (hereinafter the System) that is needed for XooNIps. 

To create XOOPS module, PHP and MySQL know-how is required.

[You can contribute on GitHub](https://github.com/XoopsDocs/XXX-tutorial). Changes will be [pushed to Gitbook.io automatically](https://xoops.gitbook.io/XXX-tutorial/activity) when the [main repository](https://github.com/XoopsDocs/XXX-tutorial) changes.
